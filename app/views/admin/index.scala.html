Opciones

A un click del empleo cuenta con una cuenta de Administrador Principal que se genera automáticamente la primera vez que se ejecuta el proyecto. Aun así, se ha habilitado la opción de crear más usuarios de administración, aunque estos últimos, cuentan con algunas limitaciones.

Configuración Amazon S3 Sólo para Administrador Principal

Para poder utilizar la plataforma, es necesario crear una cuenta en Amazon Web Services. En esta cuenta, se almacenan las fotos realizadas por los usuarios. Una vez configurada la cuenta, es necesario crear un Bucket en el servicio S3 Storage, e introducir los datos que se piden: Bucket name, Access key, Secret key.
Una vez realizada esta configuración, es necesario reiniciar el servidor para que se actualice la información y se pueda empezar a utilizar el servicio completo en la web.

¡Cuidado al cambiar la contraseña del Administrador Principal (adecco)! Si se olvida esta contraseña, es necesario contactar con el soporte para reiniciar su valor.
